# Test Commands :

First Run :

```
make
```

Second Run :

```
make BUILD_DEBUG=no
```

Third Run :

```
export BUILD_DEBUG=no
make
```

**See that just inline change will be applied !**

**Tip :** Using '@' lets to silently run command and not echoing command itself :)
